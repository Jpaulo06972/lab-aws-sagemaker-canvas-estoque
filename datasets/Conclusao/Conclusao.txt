CONCLUSÃO DO TREINAMENTO DO MACHINE LEARNING - dataset-500-curso-sagemaker-canvas-dio
A conclusão do treinamento do modelo de machine learning, desenvolvido no Amazon SageMaker Canvas para o projeto EstoqueProdutoQuantidade, revela um sistema de previsão de séries temporais com uma capacidade preditiva sólida, mas que exige atenção às configurações específicas de inventário. 
Com base no processamento do dataset de 500 linhas, o modelo alcançou um WAPE (Weighted Absolute Percentage Error) de 15,2%, o que indica uma precisão global satisfatória. Isso significa que, na maioria das previsões, o desvio entre o que o modelo previu e o que realmente ocorreu é mínimo, permitindo uma gestão de compras mais assertiva e menos baseada em suposições manuais.
Um dos insights mais relevantes obtidos durante a análise de impacto das colunas foi a neutralidade da variável FLAG_PROMOCAO. O modelo atribuiu 0% de impacto a esta coluna, sugerindo que, dentro do período analisado, as promoções não foram os principais fatores de mudança na quantidade do estoque. 
Por outro lado, a análise visual das predições individuais demonstrou que o modelo consegue captar tendências de queda acentuada com clareza, antecipando momentos de ruptura de estoque. Em cenários de alta incerteza, como observado nos intervalos de confiança superiores, o modelo sinaliza preventivamente a possibilidade de picos de demanda, permitindo que o gestor decida entre manter um estoque enxuto ou investir em uma reserva de segurança para cobrir as variações otimistas do cenário P90.
O modelo está pronto para apoiar decisões operacionais, transformando dados históricos em previsões acionáveis. A principal recomendação pós-treinamento é a monitoria contínua da métrica MAPE (29%), buscando refinar o modelo em versões futuras através da inclusão de novas variáveis sazonais ou dados de precificação.
